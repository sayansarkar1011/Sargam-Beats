# 🎵 Sargam Beats

**Sargam Beats** is a modern offline music player for Android built using **Kotlin**, **MVVM Architecture**, and **Android Jetpack**. It delivers a clean, responsive, and user-friendly music experience with smooth playback, playlist management, favorites, and background audio support.

---

## ✨ Features

* 🎶 Offline Music Playback
* ❤️ Add & Remove Favorite Songs
* 📂 Create & Manage Custom Playlists
* 🎧 Background Music Playback
* 📱 Mini Player
* 🔔 Media Style Notification Controls
* ⏯️ Play, Pause, Next & Previous Controls
* 🔀 Shuffle & Repeat Modes
* 🎵 Album Artwork Support
* 📃 Beautiful Song List UI
* ⚡ Fast & Smooth Performance
* 🌙 Material Design UI

---

## 🛠️ Tech Stack

| Category               | Technology                       |
| ---------------------- | -------------------------------- |
| **Language**           | Kotlin                           |
| **Architecture**       | MVVM (Model-View-ViewModel)      |
| **UI**                 | XML + Material Design Components |
| **Database**           | Room Database                    |
| **Concurrency**        | Kotlin Coroutines                |
| **Media Playback**     | MediaPlayer                      |
| **Background Service** | Foreground Service               |
| **Image Loading**      | Glide                            |
| **Storage**            | Room Database                    |

## 🚀 Getting Started

### Prerequisites

* Android Studio (Latest Stable Version)
* JDK 17 or above
* Android SDK

### Installation

```bash
git clone https://github.com/yourusername/Sargam-Beats.git
```

Open the project in **Android Studio**, then:

```text
1. Sync Gradle
2. Build the Project
3. Run on an Android Device or Emulator
```
