# 🎵 Sargam Beats

**Sargam Beats** is a modern offline music player for Android built using **Kotlin**, **MVVM Architecture**, and **Android Jetpack**. The application is designed to provide a clean, responsive, and user-friendly music listening experience with smooth playback, playlist management, favorites, and background audio support.

---

## ✨ Features

* 🎶 Offline Music Playback
* ❤️ Add/Remove Favorite Songs
* 📂 Create & Manage Custom Playlists
* 🎧 Background Music Playback
* 📱 Mini Player
* 🔔 Media Style Notification Controls
* ⏯️ Play, Pause, Next & Previous Controls
* 🔀 Shuffle & Repeat Modes
* 🎵 Album Artwork Support
* 📃 Beautiful Song List UI
* ⚡ Fast & Smooth Performance
* 🌙 Material Design UI

---

## 🛠️ Tech Stack

* **Language:** Kotlin
* **Architecture:** MVVM (Model-View-ViewModel)
* **UI:** XML + Material Design Components
* **Database:** Room Database
* **Asynchronous Programming:** Kotlin Coroutines
* **Dependency Injection:** ViewModel & Repository Pattern
* **Media Playback:** MediaPlayer
* **Background Service:** Foreground Service
* **Image Loading:** Glide
* **Local Storage:** Room Database

## 🚀 Getting Started

### Prerequisites

* Android Studio (Latest Stable Version)
* JDK 17+
* Android SDK

### Installation

```bash
git clone https://github.com/yourusername/Sargam-Beats.git
```

Open the project in Android Studio and run:

```bash
Sync Gradle
Build Project
Run on Device
```
